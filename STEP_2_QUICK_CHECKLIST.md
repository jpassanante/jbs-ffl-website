# Step 2: Install Dependencies - Quick Checklist

Follow these steps in order. Check each box as you complete it.

## Pre-Flight Check âœ…

- [ ] **Command Prompt/Terminal is open**
- [ ] **I'm in the fantasy-football folder** (check by typing `cd` and looking at the path)
- [ ] **I can see package.json** (type `dir` on Windows or `ls` on Mac to check)

## The Installation

- [ ] **Typed `npm install`** (exactly like that, no quotes)
- [ ] **Pressed Enter**
- [ ] **Saw text scrolling** (this is good! It's downloading)
- [ ] **Waited patiently** (2-5 minutes is normal)
- [ ] **Saw "added XXX packages"** message
- [ ] **Saw "found 0 vulnerabilities"** (or a low number)

## Verification

- [ ] **Saw a `node_modules` folder** (type `dir` or `ls` to check)
- [ ] **Saw a `package-lock.json` file** (type `dir` or `ls` to check)
- [ ] **No error messages appeared**

## If Something Went Wrong

- [ ] **Read the error message carefully**
- [ ] **Checked the detailed guide** (`STEP_2_DETAILED.md`)
- [ ] **Verified Node.js is installed** (type `node --version`)
- [ ] **Verified I'm in the right folder** (type `cd` to see current location)

---

## Common Issues Quick Fix

| Problem | Quick Fix |
|---------|-----------|
| "npm is not recognized" | Install Node.js from nodejs.org |
| "package.json not found" | Navigate to fantasy-football folder with `cd` |
| Nothing happening | Wait 2-3 more minutes, then check internet |
| Permission error | Run Command Prompt as Administrator (Windows) |
