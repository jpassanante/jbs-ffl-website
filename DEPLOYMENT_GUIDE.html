Part 1: Set Up GitHub (if you don't have an account)

1. Create a GitHub account (if needed):

  1. Go to https://github.com
  2. Click "Sign up"
  3. Follow the prompts to create your account
  4. Verify your email address

2. Install Git (if not already installed):

  1. Download from https://git-scm.com/downloads
  2. Run the installer with default settings
  3. This lets you push code to GitHub from your computer

Part 2: Create a GitHub Repository

  1. Log into GitHub and click the "+" icon in the top right
  2. Select "New repository"
  3. Fill in the details:
    • Repository name: jbs-ffl-website (or any name you like)
    • Description: "JBS Fantasy Football League Website" (optional)
    • Choose Public (required for free hosting)
    • DO NOT check "Initialize with README" (you already have code)
    • Click "Create repository"

Part 3: Push Your Code to GitHub

1. Open PowerShell in your project folder:

  • Navigate to: \\rbfil21\Users$\jpassanante\Documents\vibe-lab\fantasy-football
  • Or right-click the fantasy-football folder and select "Open in Terminal" or "Open PowerShell here"

2. Initialize Git (if not already done):

git init

3. Add all your files:

git add .

4. Create your first commit:

git commit -m "Initial commit - JBS FFL website"

5. Connect to GitHub (replace YOUR_USERNAME with your GitHub username):

git remote add origin https://github.com/YOUR_USERNAME/jbs-ffl-website.git

6. Push your code:

git branch -M main
git push -u origin main

You'll be prompted for your GitHub username and password (use a Personal Access Token, not your regular password - see note below)

Part 4: Deploy to Vercel

1. Go to Vercel:

Visit https://vercel.com

2. Sign up:

  1. Click "Sign Up"
  2. Choose "Continue with GitHub"
  3. Authorize Vercel to access your GitHub account

3. Import your project:

  1. After signing in, click "Add New..." â†’ "Project"
  2. You'll see your GitHub repositories listed
  3. Find jbs-ffl-website and click "Import"

4. Configure the project:

  • Vercel will auto-detect Next.js
  • Framework Preset: Next.js (should be auto-selected)
  • Root Directory: ./ (leave as default)
  • Build Command: npm run build (should be auto-filled)
  • Output Directory: .next (should be auto-filled)
  • Click "Deploy"

5. Wait for deployment (2-3 minutes):

  • Vercel will build and deploy your site
  • You'll see progress in real-time

6. Get your live URL:

  • Once complete, you'll see "Congratulations!"
  • Your site will be live at: https://jbs-ffl-website.vercel.app (or similar)
  • Click the URL to view your live site!

Troubleshooting

Build fails?

Check the build logs in Vercel for error messages

Can't push to GitHub?

Make sure you're using a Personal Access Token, not your password

Site not updating?

Make sure you've pushed new code to GitHub - Vercel auto-deploys on every push
